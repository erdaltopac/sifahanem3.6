# Şifahane v3.3.49

## Ayarlar ve erişilebilirlik düzenlemesi

- Ayarlar ana ekranı kategori kartlarına ayrıldı.
- Alarm, yedekleme, güvenlik, erişilebilirlik, veritabanı, yardım ve uygulama bilgileri ayrı alt ekranlara taşındı.
- Kategori kartlarının tamamı dokunulabilir hale getirildi.
- İşlem düğmeleri en az 48 dp dokunma alanına sahip olacak şekilde büyütüldü.
- Yardım konuları arama yapılabilen açılır başlıklara dönüştürüldü.
- Ayarlar bölümü için kalıcı “Büyük ayar metinleri” seçeneği eklendi.
- Ayarlar bölümü için kalıcı “Yüksek kontrastlı ayarlar” seçeneği eklendi.
- Mevcut alarm, yedekleme, güvenlik, rol ve yönetici şifresi işlevleri korundu.
- Room veritabanı sürümü değiştirilmedi.
- Kullanıcı verileri için migration gerekmemektedir.
