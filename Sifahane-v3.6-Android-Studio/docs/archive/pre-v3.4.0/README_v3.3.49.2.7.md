# Şifahane v3.3.49.2.7

## Alt menü kompakt tasarım bakım sürümü

- Menü yüksekliği ve dikey iç boşluklar azaltıldı.
- Menü öğesi genişlikleri Küçük 62 dp, Orta 74 dp, Büyük 88 dp olarak optimize edildi.
- Simge ile etiket arasındaki boşluk 1 dp seviyesine indirildi.
- Menü zemini üstte %52, altta %48 tema rengi opaklığında çok hafif dikey gradyan kullanır.
- Simge kapsülünde %10 opak siyah cam katmanı kullanılır.
- Normal simge ve etiketler %50, seçili öğeler %70 opak siyah kullanır.
- Renk geçişleri 140 ms animasyonla yumuşatıldı.
- Seçili sekmenin turkuaz vurgusu, hafif yükselmesi ve spring animasyonu korundu.
- Alt sınırda %10 → %5 → %0 siyah gradyan ile Android gezinme alanından görsel ayrım sağlandı.
- navigationBarsPadding ve 10 dp ek güvenli boşluk korundu.
- Kaydırma, sürükle-bırak, Menü/Bitti düğmesi ve boyut seçenekleri korundu.
