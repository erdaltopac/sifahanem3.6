# Şifahane v3.3.49.2.4

## Alt menü erişilebilirlik ve görsel iyileştirmeleri

- Android gezinme çubuğu güvenli alanı `navigationBarsPadding()` ile otomatik alınır.
- Güvenli alanın üstüne 10 dp ilave alt boşluk eklenmiştir.
- Üst bordür 0.5 dp kalınlığa ve düşük opaklığa indirilmiştir.
- Üst sınırda çok ince, düşük opaklıklı siyah gölge kullanılmıştır.
- Alt menü zemini %96 opaklıkla daha yumuşak bir cam hissi verir.
- Seçili sekme 1.03 ölçekli spring animasyonu ve hafif yükselme efekti kullanır.
- Seçili sekmede turkuaz vurgu ile çok hafif siyah gölge birlikte uygulanır.
- Dikey ayırıcı 0.5 dp kalınlığa düşürülmüştür.
- Kaydırma, sürükle-bırak, sabit Menü düğmesi ve Küçük/Orta/Büyük boyut seçenekleri korunmuştur.
