# Şifahane v3.3.53

Bu sürüm; grup alarmı ses/bildirim yaşam döngüsü düzeltmeleri, üç ayarlanabilir erteleme seçeneği, profil bazlı grup işlemleri, kapsamlı kullanıcı kılavuzu paylaşımı, PNG grafik paylaşımı, yedek paylaşımı, gezinme iyileştirmeleri ve merkezi Şifahane temasını içerir.

## Test notu

Gradle birim testleri, Android Lint ve debug APK derlemesi otomatik olarak çalıştırılır. Alarm, bildirim kanalı, tam ekran alarm, paylaşım hedefleri, dosya erişimi ve MediaStore davranışları farklı üretici telefonlarında gerçek cihaz testi gerektirir.
