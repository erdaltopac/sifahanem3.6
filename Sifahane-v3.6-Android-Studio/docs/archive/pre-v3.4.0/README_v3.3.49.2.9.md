# Şifahane v3.3.49.2.9

Alt menü düzenleme deneyimi iyileştirmeleri:

- Düzenleme modunda nokta/DragIndicator simgeleri kaldırıldı; her öğe kendi gerçek simgesiyle gösterilir.
- Düzenleme modundaki simgelerin çevresine düşük opaklıklı Vantablack halka eklendi.
- Sürüklenen öğenin halkası kalınlaşır ve opaklığı artar.
- Sürüklenen öğeye hafif yükselme, gölge ve %103 ölçek animasyonu uygulanır.
- Sürüklenmeyen öğelere düzenleme durumunu belirten çok hafif jiggle animasyonu eklendi.
- Sürüklenen öğe diğer öğelerin üzerinde çizilir; bırakıldığında normal görünüme yumuşak biçimde döner.
- Mevcut dinamik öğe genişliği, tam öğe hizasına oturma, kaydırma, kalıcı sıra ve Menü/Bitti işlevleri korunmuştur.
